# Git이란?

### 분산 버전 관리 시스템

- 코드의 히스토리(버전)을 관리하는 도구
- 개발되어온 과정 파악 가능
- 이전 버전과의 변경 사항 비교 및 분석



Git != GitHub

Git -> 분산 버전 관리 시스템

GitHub  -> Git 기반의 저장소 서비스

Git은 명령어를 통해서 사용
